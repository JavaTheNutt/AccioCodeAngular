Isolating Scope

This page shows how to isolate the scope directives placed on the same page

On this page there are three element directives called 'character'. Each element also has 4 custom attributes. These attributes are name, image, movetype and use-move. The character directive also has a template attached to it. This template will render a panel for each character element. The panel will display data that will be passed into it from the attributes of the element.

The panel will contain the source of the image which is passed into the img tag from the element. The same will go for the caption of the picture. Then there is a select box which will allow the user to select a move type from a list created in the controller. Then there is the input box which will allow the user to enter a move. The final element is a button which will call the useMove()function and pass the name, move type and move. This function will log a string to the screen.
