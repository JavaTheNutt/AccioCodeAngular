Directives

This shows how to create directives.

  • The first directive is an element directive. This one simply has a single template line which is printed to the screen.
  • The second directive is attached to a button and it is an attribute directive. This directive contains two functions, one to change the text of the button when the mouse enters, and another when the mouse leaves.
  • The next directive is attached to the button and it is an element directive that contains three attribute directives. Below that is an unordered list with three items. The red button has all three attributes attached and whn rolled over, will log out all three members of the shield. Each list item has one of the three attributes and will log out the respective member when rolled over.
These files can be found at:
  • HTML: 'src/partials/second_ctrl.html'
  • JS: 'src/resources/js/controllers/SecondCtrl.js'
