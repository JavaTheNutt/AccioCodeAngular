Baltimore Ravens

This page shows how to access JSON data and print it to the screen.

Here the players data is stored in a JSON file and the first 5 matching names are printed to the table using ng-repeat. This is done by creating a service in the JavaScript file which will retrieve the JSON file and pass the data to the controller which will then sort it and place it on the screen. The data can be sorted by the three fields represented through the input boxes.

These files can be accessed at:
  • HTML: 'src/partials/templates/ravens.html'
  • JS: 'src/resources/js/controllers/ravensCtrl.js'
