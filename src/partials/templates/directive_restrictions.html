Directive Restrictions

This page is used to show restrictions that can be attached to directives.

To see the output, open the console. (Ctrl, Shift, J in Chrome, Ctrl, Shift, K in FireFox). This file is located at 'src/partials/templates/directive_restrictions.html'. The JS file is located at 'src/resources/js/controllers/restrictionsCtrl.js'.

The words printed below in red are the element types that each directive is restricted to. There is also a comment restriction that cannot be seen here for obvious reasons. The messages logged to the console are attached to each directive.

These files can be found at:
  • HTML: 'src/resources/partials/directive_restrictions.html
  • JS: 'src/resources/js/controllers/restrictionsCtrl.js
