Accio code tutorial

This page is built following the tutorials on YouTube by AccioCode. The tutorials can be found here. The original source for the project can be found here. The complete source code for the site can be found here.

Each panel below corresponds to a lesson or set of lessons laid out in the course. Each panel also has a list of three attributes Route, JS File, View Template.

  • Route is the route to the view
  • JS File is the is the file that controls the particular view. These files are located at: src/resources/js/controllers/<filename>.js
  • View Template is the HTML for the particular view snippet to be loaded in. These files are located at : src/partials/templates/<filename>.html
