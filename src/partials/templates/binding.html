Binding

This page shows how binding should be utilised.

Below there is a text box and two buttons. The text box holds the value of the right button. This can be edited and the value of the button will be changed in real time. This is a good example of Angular's two-way data binding. When the button labelled 'Edit' is clicked, a second text box appears with the value of 'default'. This box controls the class attached to the second button. This file can be found at :
