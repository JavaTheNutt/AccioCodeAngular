Transclude

This page shows how to use transclude

In this page, the copyright button is hardcoded into the HTML page inside a button-directive tag. This directive has transclude set to true and uses 'data-ng-transclude' to specify where the content from the html page should be placed into the button. These files can be found at:

  • HTML: 'src/partials/templates/transclude.html'
  • JS: 'src/resources/js/controllers/transcludeCtrl.js'
