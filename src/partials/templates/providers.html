Providers

This page shows how to use custom providers

Here the background color of the box is set to black in the css. A provider is created which will be invoked by the controller. This provider contains a function to change the color of the box. The function call for this is in the .config() section of the file. This is located in the file app.js. The files can be found here:

  • HTML: 'src/partials/templates/providers.html'
  • JS: 'src/resources/js/controllers/provide.js'

The app.js file can be found here:

  • 'src/resources/js/app.js'
